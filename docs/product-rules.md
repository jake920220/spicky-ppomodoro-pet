# 제품 규칙

## 캐릭터 규칙

- 스피키는 준비된 PNG 3장만 사용한다.
- 기본 상태는 image_1이다.
- 클릭 상호작용 상태는 image_2이다.
- 타이머 종료 알림 상태는 image_3이다.

## 타이머 규칙

- 첫 버전은 뽀모도로 스타일 타이머 1개 흐름만 우선 지원한다.
- 타이머 상태는 idle / running / paused / finished 로 관리한다.
- 클릭 시 MP3 1번이 재생된다.
- 타이머 종료 시 시각적 알림과 MP3 2번 재생이 함께 발생해야 한다.

## 오디오 규칙

- 첫 버전은 준비된 MP3 2개를 사용한다.
- 오디오 재생 실패가 앱 전체 오류로 이어지면 안 된다.
- 같은 알림이 중복 재생되어 소리가 겹치지 않도록 기본적으로 방지한다. (클릭 시 MP3 1번, 타이머 종료 시 MP3 2번, 타이머 종료시에는 사용자가 직접 종료이벤트를 해지하기 전까지 디폴트 클릭 이벤트를 막는다.)

## 데스크톱 규칙

- 앱은 데스크톱 전용이다.
- 캐릭터는 사용자가 쉽게 보고 클릭할 수 있어야 한다.
- 창 동작은 macOS와 Windows에서 모두 보수적으로 구현한다.

## 비목표

- LLM 기능 없음
- 사용자 계정 없음
- 원격 서버 없음
- 클라우드 동기화 없음
- 분석 도구 없음
